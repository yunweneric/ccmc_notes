# Screenshots

This directory contains screenshots of the CCMC Notes application.

## Required Screenshots

Please add the following screenshots to this directory:

1. **home-page-desktop.png** - Desktop view of the home page showing notes library
2. **home-page-mobile.png** - Mobile view of the home page
3. **calendar-month-view.png** - Calendar month view
4. **calendar-week-view.png** - Calendar week view
5. **calendar-day-view.png** - Calendar day view with timetable
6. **pdf-viewer.png** - PDF viewer in action
7. **filters-sheet.png** - Mobile filters bottom sheet
8. **options-sheet.png** - Mobile options bottom sheet

## Screenshot Guidelines

- **Format**: PNG format
- **Resolution**: 
  - Desktop: 1920x1080 or similar
  - Mobile: 375x812 (iPhone) or similar
- **Naming**: Use kebab-case (e.g., `home-page-desktop.png`)
- **Content**: Make sure screenshots show the key features clearly
